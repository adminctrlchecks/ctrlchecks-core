# Auth0 Multi-Login Integration Guide

This folder is a complete beginner-friendly guide to add multi-login to your website using Auth0.

Goal: users can sign in with:
- Google
- Microsoft
- GitHub

## Who this guide is for

- You are new to authentication.
- You want international-standard login using OAuth2/OIDC through Auth0.
- You want one clean setup that is easier to maintain than custom auth.

## Final result you will have

- A single Auth0 login page with 3 social buttons.
- One callback flow for your app.
- User profiles created in Auth0 and mapped to your app user IDs.
- Secure session handling ready for production hardening.
